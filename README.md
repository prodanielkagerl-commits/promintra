# promintra

Este repositorio está diseñado para usarse como repositorio remoto en flujos de trabajo con Git.

## ¿Cómo usarlo como remoto?

### Agregar este repositorio como remoto en un proyecto existente

```bash
git remote add origin https://github.com/prodanielkagerl-commits/promintra.git
```

### Clonar este repositorio

```bash
git clone https://github.com/prodanielkagerl-commits/promintra.git
```

### Subir cambios al remoto

```bash
git push origin main
```

### Obtener cambios del remoto

```bash
git pull origin main
```

### Verificar los remotos configurados

```bash
git remote -v
```
